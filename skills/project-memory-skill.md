# Project memory workflow

## Goal

Keep project and client context compact, structured, traceable, and safe.

## Project session record

Record goal, completed work, changed files, checks, pending checks, risks, exact next commands, and items that must not be committed.

## Client memory

Store only operationally useful fields: identity, language, furniture type, dimensions, photos reference, style, budget, materials, deadline, area, stage, next action, and last contact time.

## Rules

- Do not store secrets or unnecessary sensitive details.
- Keep exact numbers and legal/commercial conditions unsummarized.
- Prefer structured fields over raw conversation history.
