# Furniture landing workflow

## Use when

Creating or improving a furniture-business landing page in the sites module.

## Goal

Move a visitor toward a clear inquiry while preserving the client brand and verified business facts.

## Checklist

1. State a concrete offer on the first screen.
2. Show real work or an honest placeholder state.
3. Explain furniture types, process, materials, guarantees, and next step only from approved sources.
4. Make the primary CTA explicit and mobile-friendly.
5. Test form, WhatsApp link when present, responsive layout, and generated artifact.

## Prohibited

- Invented claims, reviews, prices, guarantees, or photos.
- Generic AI-style redesign.
- Publishing without checking the target repository and deployment binding.
